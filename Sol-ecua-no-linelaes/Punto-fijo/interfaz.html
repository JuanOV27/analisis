Cómo ingresar funciones matemáticas

            
    
                
  • Para ( sin(x) ):  Math.sin(x)
    • 
                
  • Para ( cos(x) ):  Math.cos(x)
    • 
                
  • Para ( tan(x) ):  Math.tan(x)
    • 
                
  • Para logaritmo natural ( ln(x) ):  Math.log(x)
    • 
                
  • Para logaritmo base 10 ( log_{10}(x) ):  Math.log10(x)
    • 
                
  • Para ( e^x ):  Math.exp(x)
    • 
                
  • Para potencias ( x^n ):  Math.pow(x, n)
    • 
                
  • Para raíz cuadrada ( sqrt{x} ):  Math.sqrt(x)
    • 
                
  • Para raíz cúbica ( sqrt[3]{x} ):  Math.cbrt(x)
