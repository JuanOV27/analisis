Resolución de sistemas lineales

Introduce los datos en el formulario y selecciona el método de resolución.

Guía para el usuario

  • Matriz A: Introduce los valores fila por fila, separando elementos con comas y filas con un salto de línea. Ejemplo: 
    10,-1,2,0
                                -1,11,-1,3
                                2,-1,10,-1
                                0,3,-1,8
  • Vector b: Introduce los valores del vector como una lista separada por comas. Ejemplo: 
    6,25,-11,15
  • Vector inicial x0: Si no tienes un vector inicial, utiliza ceros para empezar. Ejemplo: 
    0,0,0,0
  • Tolerancia: Este valor controla la precisión del resultado. Un valor típico es 0.0001.
  • Máximo de iteraciones: Define el número máximo de pasos antes de detener el cálculo. Ejemplo: 100.
  • Método: Selecciona entre:
    • Jacobi: Usa aproximaciones previas de todas las variables.
    • Gauss-Seidel: Usa aproximaciones actualizadas en cada iteración.
