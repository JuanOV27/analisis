Método de Taylor (3 términos)

Guía de Sintaxis

  • Usa "x" para representar la variable independiente y "y" para la dependiente.
  • Operadores disponibles:
    • +: Suma
    • -: Resta
    • *: Multiplicación
    • /: División
    • ^: Potenciación (Ejemplo: x^2)
  • Funciones matemáticas soportadas: sin, cos, tan, log, exp, sqrt.
  • Ejemplo de función válida: x*y + y^2 - sin(x).
